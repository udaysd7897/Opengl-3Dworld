# Opengl-3Dworld

## Run the game
- Enter following commands in cloned directory 
1. mkdir build
2. cd build
3. cmake ../
4. make
5. Run executive files

## Controls
### Player/Boat
- Shoot "x"
- Right "Right arrow key"
- Left "Left arrow key"
- Jump "Space key"

### Camera views
- boatview "b"
- cameraview "c"
- towerview "t"
- helicopterview "h" + scroll with mouse to hover around
